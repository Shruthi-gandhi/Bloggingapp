## 🚀 Features

- ✍️ Create, Read, Update, and Delete blog posts

- 📄 Rich text editor for blog content
- 📦 RESTful APIs to handle blog operations
- 🧑 Profile management
- 🌙 Responsive design with a clean UI

### 📌 Client (Frontend)
- React.js
- Axios
- React Router DOM
- Tailwind CSS (or Bootstrap, if used)
- HTML/CSS

### 📌 Server (Backend)
